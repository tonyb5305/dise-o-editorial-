# Manifiesto de Diseño – Luis

Este proyecto consiste en la creación de un sitio web interactivo que explora los principios fundamentales del diseño visual a través de una experiencia moderna y estructurada.

## 🎯 Objetivo del Proyecto

Desarrollar una página web tipo portafolio conceptual que combine:

- Diseño editorial moderno
- HTML5 semántico
- CSS Grid
- Animaciones con JavaScript
- Experiencia de usuario fluida

El sitio presenta cinco pilares del diseño:

- Minimalismo
- Contraste
- Tipografía
- Composición
- Color

Cada sección incluye explicación conceptual e imagen representativa.

---

## 🧩 Estructura del Proyecto

El proyecto está dividido en:

- `index.html` → Estructura semántica del sitio.
- `styles.css` → Estilos visuales y animaciones.
- Google Fonts → Tipografía estética y legible.

---

## 🎨 Características Implementadas

- Scroll suave entre secciones.
- Animaciones al aparecer cada concepto.
- Diseño alternado imagen/texto.
- Tarjetas interactivas con efecto hover.
- Botón flotante para volver al inicio.
- Footer con derechos de autor personalizados.

---

## 💡 Tecnologías Utilizadas

- HTML5
- CSS3
- CSS Grid
- Variables CSS
- JavaScript (animaciones básicas)
- Google Fonts
